CMD中无法使用node及npm

# 无法使用 node 这个原因一般是环境配置有误 一般情况下，安装 nodejs 会自动添加环境变量 如果出错应该先检查环境变量 然后将 Path 内 nodejs 路径提至最前 # 无法使用 npm 这次安装 nodejs 后，在 cmd 中， node -v 可以显示结果。 但是， npm -v 显示为空。 最后重新卸载、安装 nodejs 还是没有解决。 最后解决的方法是：发现在用户变量中，存在一个路径： C:\Users\dname\AppData\Roaming\npm 应该是之前安装上去的一个文件，把这个 npm 文件夹删除后，成功解决问题。
