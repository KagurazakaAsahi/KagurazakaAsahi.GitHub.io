上网冲浪，一个优秀的输入法可以大大提升舒适度，因此咱想分享下咱现在使用的输入法

# 电脑输入法

# 选择理由

Windows 系统自带的输入法确实还可以，虽然没有皮肤，但好在外形简洁，不容易腻.

但是该输入法还是存在不少 bug , 平常使用不太能感受到，但遇上时那感觉是十分难受的.

具体是什么 bug 这里就不多赘述.

在试了网上诸多输入法后咱还是选择了搜狗输入法.

作为一款老牌中文输入法，它的不存在什么 bug, 功能十分完善.

唯一的缺点就是广告了.

这里咱去去网上找到了一个免安装去广告的精简版，

那感觉一下子就上来了.

并且该输入法还有最多数量的皮肤，这就太舒服的.

现在不喜欢花里胡哨了，咱就挑了个比较简洁的.

一个字，爽.

# 个人配置

  • 修改字体为 MonaFang SC (个人用 Manaco 与 PingFang SC Medium 魔改的字体😋)
  • 修改皮肤为 【十月】线交集 (一款简洁风的皮肤)
  • 添加自定义短语如: qqm (QQ 邮箱), dh (电话号码), web (个人网站) 等
  • 显示候选词改为 8 个，原始的 5 个太短 (任性，谁叫咱有 4k 屏呢

# 手机输入法

# 选择理由

手机，作为隐私泄露的重要源头之一，咱还是对国产输入法抱有警戒之心.

因此，咱选择了大名鼎鼎的谷歌出品的 Gboard ,

别的不说，就冲它设置多，没啥花里胡哨的功能就给个好评.

# 个人配置

  • 添加语言布局 (咱还是时不时输点日语的)

  • 改个主题背景

  • 字典增加自定义短语

  • 将触感调到适中 (这个输入法能调触感太关键了)
