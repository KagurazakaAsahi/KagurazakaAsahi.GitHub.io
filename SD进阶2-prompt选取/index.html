# 前言

在使用 SD 来 AI 绘图的过程中， prompt 的选取是至关重要的，毕竟这是决定绘制的图像主题的决定性因素，

经过咱的大量实验，将 prompt 大致分为四类:

  • 基础：图片本身的效果，包括清晰度，图片种类 (水墨，油画，手绘...), 画风等
  • 背景：人物所在环境，季节，早晚，景物等
  • 人物：人物本身特征，性别，衣着，表情，动作等
  • 设定：图片构图等一些细节的设定

每次在使用 prompt 时，按顺序将上述每类填写完毕即可初步绘制一个较好的图片.

与此同时，由于分类后， prompt 组合的泛用性大大增加，因此如果有较好的 prompt 组合可以记录下来，

不同种类的 prompt 组合相互搭配可以大大减少工作量，避免每次从头重新输入 prompt
