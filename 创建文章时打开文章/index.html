在 hexo 中，常常使用 hexo n '文章名字' 来创建文章，可是该命令创建完文章后并不会打开，仍需到 source 文件夹找到新创建的文章打开编辑。这十分不方便.

作为懒癌，肯定不能就这样.

根据 hexo 官方文档，解决办法很简单，在 root 文件夹下新建一个 scripts 文件夹，然后在该文件夹下创建一个 名字任意的.js 文件即可.
