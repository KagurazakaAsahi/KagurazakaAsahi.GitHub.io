# 建站原因

  • 建站第一原因就是好玩啦，咱就是个爱折腾的人，建一个属于咱自己的网站多么酷啊！
  • 其次就是存放些学习资料 (是正经学习资料！) 和笔记，把一些笔记放博客上，随时都能看，也不怕丢失.
  • 再然后就是写点日常不发生的事，权当日记了，记录一下这美(hei)好(an)的生活.
  • 最后...... 好像没啥原因了...... 之后再补！
