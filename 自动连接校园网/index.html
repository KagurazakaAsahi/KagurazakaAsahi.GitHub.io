# 碎碎念

学校的校园网是哪个 sb 想出来的，联网还得先登录。一天一次花 10s , 一年就是 3650s 也就是将近 1 个小时花在联网上。咱早就想解决这个问题了。不过其实也简单，一个 JS 脚本就足够了，而且别人已经有基础的代码了，咱就在此基础上改进改进，做到全自动连接校园网。这里咱以南昌大学的深澜软件系统为例，来实现开机自动连接校园网.

# 实现方式

  1. 使用 JS 脚本，打开校园网登录网页时自动输入账号密码，并登录，然后关闭网页
  2. 将浏览器启动的默认标签页改为校园网登录网页
  3. 开机自启浏览器

# 具体步骤

# 安装脚本

Tampermonkey - Microsoft Edge Addons

首先安装上述油猴脚本管理器.

在安装以下脚本

南昌大学校园网自动连接 - 改 (greasyfork.org)

然后浏览器右上角找到油猴 -> 管理面板

找到刚才安装的脚本进行编辑

输入自己相应的信息即可

修改完记得左上角文件 -> 保存

完成，现在再次进入校园网登录界面就可以自动登录了

# 设置浏览器启动默认标签页

由于只有第一次登录需要到登录界面，因此不能直接在浏览器内修改每一次的默认标签页.

这里咱们先新建一个浏览器快捷方式.

在资源管理器输入 C:\Program Files (x86)\Microsoft\Edge\Application

找到 msedge.exe 右键新建快捷方式到桌面.

右键属性，在目标一栏加上 http://222.204.3.154/ (注意网址前有个空格)

现在点击这个快捷方式就可以联网并自动关闭了.

# 将浏览器加入开机自启

最后一步了，

为了每天自动启动该快捷方式，将其加入开机自启，方法如下:

将该快捷方式复制到资源管理器下

C:\ProgramData\Microsoft\Windows\Start Menu\Programs\StartUp 即可

# 完成

现在可以摆脱每天开机第一件事就是联网了，简直不要太舒服！！！

其他学校软件软件或其他浏览器也类似，咱就不一一赘述了.

半个小时改脚本，之后都不用自己联网了，节省一大笔时间，血赚！！！
