# 为什么使用 Git

在编写代码过程中，经常需要修改代码

并且创建不同版本

例如

  • 第一版.cpp
  • 第二版.cpp
  • 第三版.cpp
  • 最终版.cpp
  • 最终版 2.0.cpp
  • 最终版不修改版.cpp
  • ......

但是，修改过程中难免会有不满意的地方

这时，需要回滚到未修改之前的状态

当代码数量少时可以使用撤回快捷键

但当数量太大时，这个操作就很鸡肋了

这时我们就要用上 Git 这个工具了

使用让 Git 帮助我们管理代码版本
